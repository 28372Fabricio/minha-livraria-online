# 📚 Livraria Online

## 📝 Descrição

A **Livraria Online** é uma aplicação web desenvolvida para facilitar a compra e venda de livros pela internet. O sistema permite que usuários naveguem por um catálogo de livros, realizem buscas, adicionem itens ao carrinho e finalizem compras de forma simples e intuitiva.

---

## 🚀 Funcionalidades

* 📖 Listagem de livros disponíveis
* 🔍 Busca por título, autor ou categoria
* 🛒 Carrinho de compras
* 💳 Finalização de pedidos
* 👤 Cadastro e login de usuários
* ⭐ Avaliação e comentários de livros (opcional)

---

* **Frontend:** HTML, CSS, JavaScript

## ⚙️ Como Executar o Projeto

### Pré-requisitos

* Node.js instalado
* Gerenciador de pacotes (npm ou yarn)
* Banco de dados configurado

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/livraria-online.git

# Acesse a pasta do projeto
cd livraria-online

# Instale as dependências
npm install

# Inicie o servidor
npm start
```

---

## 🔐 Variáveis de Ambiente

Crie um arquivo `.env` na raiz do projeto e configure:

```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=sua_senha
DB_NAME=livraria
PORT=3000
```
